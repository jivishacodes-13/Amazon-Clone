## 🔧 Customization

### Change Product Data
Edit `src/components/Products.jsx`:
```jsx
const productSections = [
  {
    id: 1,
    title: 'Your Category',
    products: [
      { id: 1, name: 'Product Name', image: 'url', price: '999' },
      // Add more...
    ]
  }
];
```

### Change Colors
Edit `tailwind.config.js`:
```js
colors: {
  amazon: {
    darkNav: '#YOUR_COLOR',
    orange: '#YOUR_ORANGE',
    // etc.
  }
}
```

### Modify Navbar
Edit `src/components/Navbar.jsx`:
- Change logo text
- Add/remove categories
- Modify search placeholder
- Update delivery location
